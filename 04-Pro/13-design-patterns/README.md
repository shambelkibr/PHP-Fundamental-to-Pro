# Design Patterns in PHP

## Introduction

Design patterns are reusable solutions to common problems in software design.

## Topics Covered

### 1. Creational Patterns
- Singleton
- Factory
- Builder

### 2. Structural Patterns
- Adapter
- Decorator
- Facade

### 3. Behavioral Patterns
- Observer
- Strategy
- Command

### 4. Architectural Patterns
- MVC (Model-View-Controller)
- Repository Pattern

## Examples

- `01-singleton.php` - Singleton pattern
- `02-factory.php` - Factory pattern
- `03-observer.php` - Observer pattern
- `04-mvc-example/` - MVC implementation

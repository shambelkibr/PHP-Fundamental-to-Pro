# Object-Oriented Programming Basics

## Introduction

OOP is a programming paradigm that uses "objects" to design applications and programs.

## Topics Covered

### 1. Classes and Objects
- Defining classes
- Creating objects
- Properties and methods

### 2. Constructor and Destructor
- __construct() method
- __destruct() method
- Initialization

### 3. Access Modifiers
- public
- private
- protected

### 4. Inheritance
- Extending classes
- Parent and child classes
- Method overriding

### 5. Special Methods
- Magic methods
- Static methods
- Abstract classes

## Examples

- `01-classes-objects.php` - Basic OOP concepts
- `02-constructor.php` - Constructors and initialization
- `03-inheritance.php` - Class inheritance
- `04-access-modifiers.php` - Visibility control

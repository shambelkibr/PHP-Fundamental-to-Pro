# Database Operations with PDO

## Introduction

Learn how to interact with databases using PHP Data Objects (PDO) - the modern, secure way to work with databases in PHP.

## Topics Covered

### 1. Database Connection
- PDO connection
- Connection options
- Error handling

### 2. CRUD Operations
- Create (INSERT)
- Read (SELECT)
- Update (UPDATE)
- Delete (DELETE)

### 3. Prepared Statements
- Security benefits
- Parameter binding
- Named vs positional placeholders

### 4. Advanced Queries
- JOIN operations
- Transactions
- Fetch modes

## Examples

- `01-connection.php` - Connecting to database
- `02-crud-operations.php` - Basic CRUD
- `03-prepared-statements.php` - Secure queries
- `04-transactions.php` - Transaction handling

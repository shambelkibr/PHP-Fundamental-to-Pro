## ğŸš€ Getting Started

### Prerequisites
- PHP 7.4 or higher
- Web server (Apache/Nginx) or PHP built-in server
- MySQL (for database lessons)
- Composer (for advanced topics)

### Installation
```bash
# Clone the repository
git clone https://github.com/shambelkibr/PHP-Fundamental-to-Pro.git

# Navigate to the directory
cd PHP-Fundamental-to-Pro

# Start PHP built-in server (optional)
php -S localhost:8000
```

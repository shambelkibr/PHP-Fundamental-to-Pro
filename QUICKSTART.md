# Quick Start Guide

Get started with PHP learning in just a few minutes!

## Prerequisites

You need PHP installed on your system. Check if it's installed:

```bash
php --version
```

If not installed, download from [php.net](https://www.php.net/downloads.php)

## Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/shambelkibr/PHP-Fundamental-to-Pro.git
cd PHP-Fundamental-to-Pro
```

### 2. Start Learning

Begin with the Beginner section:

```bash
cd 01-Beginner/01-basics
```

### 3. Running PHP Files

#### Method 1: Direct Execution
```bash
php 01-hello-world.php
```

#### Method 2: Built-in Web Server
```bash
# Start server in the project root
php -S localhost:8000

# Then open your browser to:
# http://localhost:8000/01-Beginner/01-basics/01-hello-world.php
```

### 4. Try the Todo App Project

```bash
cd 05-Projects/01-todo-app
php -S localhost:8000
```

Then open http://localhost:8000 in your browser.

## Learning Path

### Week 1-2: Beginner (01-Beginner)
1. **Basics** - Variables, data types, operators
2. **Control Structures** - If/else, loops, switch
3. **Functions** - Creating reusable code
4. **Arrays** - Working with collections

### Week 3-4: Intermediate (02-Intermediate)
1. **OOP Basics** - Classes and objects
2. **File Handling** - Reading and writing files
3. **Forms** - Processing user input
4. **Sessions & Cookies** - Managing state

### Week 5-7: Advanced (03-Advanced)
1. **Database** - MySQL with PDO
2. **Error Handling** - Exceptions and debugging
3. **Security** - Best practices and protection
4. **API Development** - Building RESTful APIs

### Week 8-10: Pro (04-Pro)
1. **Design Patterns** - Professional code organization
2. **Testing** - PHPUnit and TDD
3. **Composer** - Dependency management
4. **Frameworks** - Laravel and Symfony

## Common Commands

```bash
# Check PHP syntax
php -l filename.php

# Run PHP file
php filename.php

# Start web server
php -S localhost:8000

# Check PHP info
php -i

# Interactive PHP shell
php -a
```

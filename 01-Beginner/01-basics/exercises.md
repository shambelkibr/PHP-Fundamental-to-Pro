# Exercises - PHP Basics

Practice these exercises to reinforce your understanding of PHP basics.

## Exercise 1: Personal Information
Create a PHP script that:
- Declares variables for your name, age, city, and favorite programming language
- Displays this information in a formatted way using echo
- Uses at least 3 different data types

## Exercise 2: Calculator
Create a simple calculator that:
- Defines two number variables
- Performs all arithmetic operations (+, -, *, /, %)
- Displays the results with appropriate labels

## Exercise 3: Temperature Converter
Create a script that:
- Defines a temperature in Fahrenheit as a constant
- Converts it to Celsius using the formula: C = (F - 32) Ã— 5/9
- Displays both temperatures

## Exercise 4: Type Checking
Create a script that:
- Declares variables of different types (string, integer, float, boolean, array)
- Uses type checking functions to verify each variable's type
- Displays the results in a readable format

## Exercise 5: Comparison Practice
Create a script that:
- Defines two variables with the same numeric value but different types (int and string)
- Tests all comparison operators (==, ===, !=, !==, >, <, >=, <=)
- Explains the difference between == and ===

## Exercise 6: Magic Constants
Create a script that:
- Uses at least 5 magic constants
- Displays their values
- Creates a function that shows __FUNCTION__ and __LINE__

## Exercise 7: String Operations
Create a script that:
- Defines your first and last name as separate variables
- Concatenates them with a space
- Converts the result to uppercase
- Displays the length of the full name

## Exercise 8: Boolean Logic
Create a script that simulates a simple login system:
- Define constants for username and password
- Create variables to check if username is correct AND password is correct
- Use logical operators to determine access
- Display appropriate message

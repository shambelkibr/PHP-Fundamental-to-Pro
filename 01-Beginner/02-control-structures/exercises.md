# Exercises - Control Structures

Practice these exercises to master control structures in PHP.

## Exercise 1: Grade Calculator
Create a script that:
- Takes a score (0-100)
- Uses if...elseif...else to assign letter grades:
  - A: 90-100
  - B: 80-89
  - C: 70-79
  - D: 60-69
  - F: Below 60
- Displays the score and grade

## Exercise 2: Day of Week
Create a script using switch that:
- Takes a day number (1-7)
- Displays the day name
- Indicates if it's a weekday or weekend

## Exercise 3: Even Numbers
Create a script that:
- Uses a for loop to display even numbers from 1 to 20
- Calculates and displays their sum

## Exercise 4: Multiplication Table
Create a script that:
- Uses nested loops to generate a multiplication table (1-10)
- Formats the output in a table structure

## Exercise 5: Pattern Generator
Create a script that generates this pattern using loops:
```
*
**
***
****
*****
```

## Exercise 6: Number Guesser
Create a script that:
- Has a secret number (1-100)
- Uses a while loop to "guess" numbers
- Uses break when the correct number is found
- Counts how many attempts it took

## Exercise 7: Array Filter
Create a script that:
- Has an array of numbers (including negative and zero)
- Uses foreach with continue to display only positive numbers
- Calculates the sum of positive numbers

## Exercise 8: Prime Numbers
Create a script that:
- Finds and displays all prime numbers between 1 and 50
- Uses nested loops and break/continue appropriately

## Exercise 9: Login System
Create a simple login system that:
- Has predefined username and password
- Uses a while loop to allow 3 login attempts
- Uses break on successful login
- Displays appropriate messages

## Exercise 10: Menu System
Create an interactive menu system that:
- Displays a menu of options (1-5)
- Uses switch to handle each choice
- Uses a loop to keep showing the menu
- Option 5 exits the program

## Solutions

Try to solve these on your own first! Solutions are available in the `solutions/` directory.
